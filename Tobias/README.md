To activate the virtual environment, you need to enable script execution in PowerShell by setting the execution policy.
Open PowerShell as Administrator.
Run the following command to allow script execution: `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned`.
Confirm the change by typing `Y` and pressing Enter.
Now, you can activate the virtual environment using the command: `.venv\Scripts\activate.ps1`.
